Flex

Pode-se atribuir valores as propriedades grow, shrink e basis utilizando flex.

Os valores devem seguir a ordem das propriedades. flex: grow shrink basis

Exemplo: flex: 1 0 auto;

No exemplo acima grow recebeu 1, shrink recebeu 0 e basis recebeu auto.

flex: 1; -> (grow = 1, shrink = 1, basis = 0)
